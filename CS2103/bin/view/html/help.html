J.Listee Your Personal To-Do Manager

Features and Commands

J.Listee is a GTD desktop application which can help you to manage your to-do list.

Features include:
♦  create a task with or without date and time
♦  edit an existing task
♦  delete existing tasks
♦  search tasks by keywords
♦  postpone a task

# Command format Example
1 add <task description> [([<start time> -] <end time>)] [@ location] [#<tag>[#<tag> […]]] add Computer class (every Friday 16:00-18:00) @icube #lecture
2 update <task number> [<new task description>] [- ()] [([<new start time> -] <new end time>)] [-@] [@ new location] [-#<existing tag> [#<new tag> […]]] update 1 SE lecture
update 2 -@
